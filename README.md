# tetris-html

Tetris built with html, css, and js.

Project started on 7 June 2024.

===========================================================================

## Editable variables in customisation.js:
<ul>
  <li>Grid dimensions:
    <ul>
      <li>rowHeight</li>
      <li>columnWidth</li>
    </ul>
  </li>
  <li>Pre-game countdown:
    <ul>
      <li>countdownDuration</li>
    </ul>
  </li>
  <li>Block aesthetics:
    <ul>
      <li>colors</li>
    </ul>
  </li>
</ul>
